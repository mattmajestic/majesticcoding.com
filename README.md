# majesticcoding.com Documentation

Welcome to majesticcoding.com! 

## Run with Kubernetes 📦
```
kubectl apply -f k8s-go.yaml
```

## Run with Docker 🐳
```
docker compose up
```

## Run with Terraform 🌍
```
go run terraform.go
minikube service majesticcoding-service
```
## Run with Go 🐹
```
go run cmd/main.go
```
